## Legend

| Symbol | Meaning |
|--------|---------|
| `─────►` | Data flow direction |

| `◄────►` | Bidirectional communication |
| `▼` | Flow continuation |
| `│` | Vertical flow |
| `┼` | Junction/branch point |
