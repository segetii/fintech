Compliance Decision Matrix — Deterministic Action Resolution

risk < 200 ∧ ¬sanctioned ∧ geo ∉ {PROHIBITED} → APPROVE (minimal)
200 ≤ risk < 400 ∧ ¬sanctioned → APPROVE (low)
400 ≤ risk < 600 ∧ ¬sanctioned → REVIEW (medium)
600 ≤ risk < 800 → ESCROW (high)
risk ≥ 800 ∨ ≥4 red flags → BLOCK (critical)
sanctioned ∨ FATF_blacklist → BLOCK (absolute)
geo ∈ {VERY_HIGH, HIGH} → ESCROW (geo override)
risk ≥ 80 ∨ action = BLOCK → SAR filing triggered
RiskResponse → action ∈ {APPROVE, REVIEW, ESCROW, BLOCK, REQUIRE_INFO}
